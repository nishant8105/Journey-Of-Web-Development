### CSS variables
- Css variables are used to set values in variables to change in single click

- Variables are set in :root block with two -- sign before variable name

- var() is use to use variable

### Media Queries
Media queries are used in CSS to apply styles based on device
characteristics like screen width, height, resolution, or orientation.

They are mainly used to build responsive websites.
