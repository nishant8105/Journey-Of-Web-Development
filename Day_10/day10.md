Today I have learn About CSS(Cascading Style Sheet).
CSS is use to give styleing to skeleton of HTML.
There are 3 type of CSS:
<ol>
<li> Inline Css
<li>Internal Css
<li> External Css
</ol>  

- Inline Css : Giving style in side the HTML tag using style attribute
- Internal Css : Giving style in side the HTML file using &lt;Style&gt;
- External Css : Giveing style by crating .css file and link to HTML file through &lt;link&gt;
